# logika-javascript
Solutions for 5 logic and algorithmic challenges in JavaScript, covering Queue Management, Balanced Brackets, Meeting Rooms Scheduling, LRU Cache, and N-ary Tree LCA.

# Solusi 5 Soal Logika JavaScript

Repository ini berisi penyelesaian untuk 5 soal logika algoritma menggunakan bahasa pemrograman JavaScript.

## Daftar Soal
1. Antrian Kasir Supermarket
2. Validasi Tanda Kurung (Balanced Brackets)
3. Alokasi Ruangan Rapat Minimum (Meeting Rooms)
4. LRU Cache (Least Recently Used)
5. Lowest Common Ancestor (LCA) pada N-ary Tree

## Cara Menjalankan
Pastikan Anda sudah menginstall [Node.js](https://nodejs.org/). 
Jalankan file menggunakan terminal:

node soal1.js
node soal2.js
node soal3.js
node soal4.js
node soal5.js
