# logika-javascript
Solutions for 5 logic and algorithmic challenges in JavaScript, covering Queue Management, Balanced Brackets, Meeting Rooms Scheduling, LRU Cache, and N-ary Tree LCA.

# Solusi 5 Soal Logika JavaScript

Repository ini berisi penyelesaian untuk 5 soal logika algoritma menggunakan bahasa pemrograman JavaScript.

## Daftar Soal
1. Antrian Kasir Supermarket
2. Validasi Tanda Kurung (Balanced Brackets)
3. Alokasi Ruangan Rapat Minimum (Meeting Rooms)
4. LRU Cache (Least Recently Used)
5. Lowest Common Ancestor (LCA) pada N-ary Tree

## Daftar Masalah & Solusi
### 1. Antrian Kasir Supermarket (`soal1.js`)
Menentukan kasir mana yang akan melayani pelanggan berdasarkan antrian terpendek.
- **Konsep:** Array, Indexing, dan Pencarian Nilai Minimum.

### 2. Validasi Tanda Kurung Seimbang (`soal2.js`)
Memvalidasi apakah string mengandung tanda kurung `()`, `[]`, dan `{}` yang seimbang dan bersarang dengan benar.
- **Konsep:** Data Structure (Stack).

### 3. Alokasi Ruangan Rapat (`soal3.js`)
Menghitung jumlah minimum ruangan yang dibutuhkan agar jadwal rapat tidak ada yang bertabrakan.
- **Konsep:** Sorting dan Two Pointers (Chronological Sorting).

### 4. LRU Cache (Least Recently Used) (`soal4.js`)
Membangun sistem cache dengan kapasitas terbatas yang membuang item yang paling jarang digunakan.
- **Konsep:** Data Structure (Map) dengan kompleksitas waktu $O(1)$.

### 5. Lowest Common Ancestor - N-ary Tree (`soal5.js`)
Mencari atasan bersama terdekat (LCA) dari dua karyawan dalam struktur organisasi berbentuk pohon.
- **Konsep:** Tree Traversal dan Rekursi (DFS).
